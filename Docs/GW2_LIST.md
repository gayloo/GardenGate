# MiscSettings
| Setting Description                                                                 | Internal Name                                          |
|-------------------------------------------------------------------------------------|--------------------------------------------------------|
| Enables janky functionality for Friendly Fire                                       | SyncedGame.EnableFriendlyFire                          |
| Sets the team you spawn in (Zombies = 1, Plants = 2). Used in modes like Boss Hunts and Infinity Time | GameMode.ModeTeamId                  |
| Sets the difficulty (Multiplayer = 0, Easy = 1, Normal = 2, Hard = 3, CRAAAZZY = 4) | GameMode.StoredDifficultyIndex                         |
| Sets what characters are available using their Primary Weapon Unlock Asset Identifier (separated by `;`)        | GameMode.AvailableCostumes |

e.g. GameMode.AvailableCostumes 1364115386;-1690576028;-1633673158  - Allows only Default All-Star, Imp, and Citron to be selected.
